Full Stack Developer
--------------------

Hey, I'm Stelios — a Full Stack Developer with an unconventional career path.

I started with network and telecommunications studies, which gave me a solid understanding of how the internet works before I ever wrote a line of code. Then I was a taxi driver for years, which, as it turns out, is a surprisingly good teacher of patience, adaptability, and problem-solving under pressure.

A coding bootcamp leaflet handed to me mid-shift changed everything. A few years later, here I am.

I build and maintain e-commerce platforms using ASP Classic, MS SQL Server, JavaScript, CSS, and IIS.

* 🌍  I'm based in Thessaloniki, Greece
* ✉️  You can contact me at [stysns@outlook.com](mailto:stysns@outlook.com)
